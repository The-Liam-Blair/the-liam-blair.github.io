Honours Project

University honours project that investigated using the Wave Function Collapse algorithm and Voronoi Regions to generate 2D terrain

GitHub

Serving as my honours project for my final year in university, the aim of the project was to investigate the use of Wave Function Collapse (WFC) and its uses in procedurally generating 2D terrain, specifically on increasing the amount of control the developer has over the procedural generation.

The project culminated with combining the WFC algorithm with Voronoi Regions, where Voronoi Regions would be used to generate 'Biomes', or designated areas within the 2D map that dictates the tile that is allowed to generate there. Additionally Lloyd Relaxation can be applied to more unify the regions.
Afterwards, the WFC algorithm would detect and correct biomes such that all adjacent biomes are valid, and invalid biomes would be replaced to satisfy the WFC algorithm's adjacency ruleset.
Finally, the WFC algorithm would apply a custom-made 'Biome Spread', a configurable value per tile type that allowed biomes to encroach and take over each other's tiles to more seamlessly blend biomes together.

Ultimately the project delivered a combined WFC and Voronoi Regions algorithm that allowed some additional control of generation 2D maps; utilising six different tile/biome types to generate simple but varied maps with extensive user input to finely control the generation of each map.
